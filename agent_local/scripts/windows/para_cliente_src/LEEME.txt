CONTENIDO DE ESTE ZIP:
  INSTALAR.bat          → ejecutar una sola vez para instalar
  Iniciar Conector.bat  → ejecutar todos los dias para arrancar
  python\               → Python incluido (no necesitas instalarlo)
  app\                  → código del conector (no tocar)

----------------------------------------------------------------
INSTALACION (solo una vez):
----------------------------------------------------------------

  1. Extraer todo el contenido del zip a una carpeta
     (por ejemplo: C:\ConectorCirugias\)

  2. Doble clic en INSTALAR.bat

  3. Esperar que termine (necesita internet, ~5 minutos)

  4. Al finalizar aparece el icono "Iniciar Conector"
     en tu Escritorio

  LISTO. Ya no necesitas hacer nada mas.

----------------------------------------------------------------
USO DIARIO:
----------------------------------------------------------------

  1. Doble clic en "Iniciar Conector" del Escritorio
  2. Aparece una ventana negra con el texto "CONECTOR ACTIVO"
     DEJARLA ABIERTA mientras trabajas
  3. Usar la pagina web normalmente

  Para cerrar el conector: cerrar esa ventana negra

----------------------------------------------------------------
PRIMERA VEZ CON RECETAS O LENTESS:
----------------------------------------------------------------

  Cuando ejecutas Recetas o Lentess por primera vez:
  - Chrome se abre automaticamente en PAMI
  - Haces login vos (usuario, contrasena, OTP si pide)
  - El sistema detecta el login y continua solo
  - La proxima vez ya NO necesita login de nuevo

----------------------------------------------------------------
PROBLEMAS COMUNES:
----------------------------------------------------------------

  "Conector: no detectado" en la pagina web
  → Abre el "Iniciar Conector" del Escritorio

  Chrome no abre al ejecutar Recetas/Lentess
  → Verifica que la ventana negra del conector este abierta

  Error "puerto 8765 en uso"
  → El conector ya esta corriendo, no hace falta abrirlo de nuevo

  Para soporte: contactar al administrador del sistema
